# Upstream limitations

!!! info "Inventory context"
This page is checked against upstream `sysml-2ls` package `0.9.1` at revision `a0b3ddbf783063dd7291aac0b51d4282decc789e`.

The upstream core provides syntax, linking, and semantic validation. It is not a temporal model checker. State-machine-related validators are structural SysML checks exposed through ordinary diagnostics.

Inventory pages distinguish static discovery from dynamic reproduction. Static inventory rows should not be read as a guarantee that every diagnostic has a stable service-level reproduction fixture.
